# README

## Dependencies

Generate 'requirements.txt'

```bash
# pip install --user pipreqs
pipreqs ./
```

Using VENV
[documenation](https://docs.python.org/3/library/venv.html)
```bash
# Create virtual env
python3 -m venv venv --without-pip --system-site-packages

# Activate
source ./venv/bin/activate
```
